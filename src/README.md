# Radio – Proyecto de Algoritmos y Estructura de Datos

Este proyecto implementa una simulación de una radio de carro utilizando el lenguaje Java.
La radio permite encenderse y apagarse, cambiar entre AM y FM, subir y bajar frecuencias,
guardar emisoras en botones y seleccionar emisoras guardadas.

El programa fue desarrollado siguiendo el uso de interfaces para permitir que diferentes
implementaciones de radio puedan ser utilizadas sin modificar el programa principal.

---

## Funcionalidades
- Encender y apagar la radio
- Cambiar entre modo AM y FM
- Subir y bajar la frecuencia
- Guardar frecuencias en botones (1 al 12)
- Seleccionar frecuencias guardadas
- Mostrar el estado actual de la radio

---

## Compilación
Desde la carpeta raíz del proyecto, ejecutar el siguiente comando:

javac Main.java

---

## Ejecución
Luego de compilar, ejecutar:

java Main.java
